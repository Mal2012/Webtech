Ausgabemedien im Stylesheet berücksichtigen

Grundsätzliches

Mit dem Attribut media beim Element link können verschiedene Ausgabemedien berücksichtigt werden. Mit dem Attributwert screen wird die Darstellung der Webseite auf dem Bildschirm gesteuert. Die Ausgabe auf dem Drucker kann mit dem Attributwert print geregelt werden.

Implementieren und Testen

Zum Test eines Stylesheets für die Ausgabe auf dem Drucker kann die Druckervorschau verwendet werden. Allerdings werden in der Druckvorschau und auch beim Druck keine Hintergrundfarben oder Hintergrundelemente wie Bilder in der Voreinstellung von Firefox 3.0 dargestellt. Es gibt bei Firefox 3.0 eine Option unter Seite einrichten: Hintergrund drucken (Farben und Bilder).

Beispiel

Im vorliegenden Beispiel sind in der Browseransicht keine Wasserzeichen zu sehen. Wenn Sie die Seite ausdrucken wollen, dann sollen Smileys als Wasserzeichen gedruckt werden. Schauen Sie sich die Druckvorschau an. Die Smileys sollen auch gedruckt werden, wenn diese Option im Firefox ausgeschaltet ist!
Außerdem soll die Naviagationsleiste, im Browser auf der linken Seite zu sehen, nicht mit gedruckt werden.

Desweiteren sollen im Druck die URLs zu den Hyperlinks erscheinen, die in der Browseransicht nicht angezeigt werden.

Literatur und Referenzen

Für weitere Details wird auf die CSS-Spezifikation 2.1 (http://www.w3.org/TR/CSS2/) verwiesen.

Die Möglichkeit Ausgabemedien zu benutzen, ist in der Spezifikation zu CSS 1.0 (http:////www.w3.org/TR/REC-CSS1/) nicht vorhanden.
