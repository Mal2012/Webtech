Stylesheets:
Verwenden von Klassen, kontextabhängigen Selektor und individuellen Selektoren

1. Für das Element body sind die Schriftart auf "helvetica" und die Hintergrundfarbe auf "#e0e0e0" einzustellen.

2. Für das Element h1 sind die Schriftfarbe blau und die Schriftgröße auf 16pt einzustellen.

3. Für das Element p ohne Zusatzangaben wie Kontext oder Klasse sind die Voreinstellungen zu lassen.

4. Dieser Paragraph ist von der Klasse mono. Für solche Paragraphen ist die Schriftart monospace einzustellen.

5. Dieser Paragraph ist von der Klasse rot. Für alle Elemente der Klasse rot ist die Schriftfarbe rot einzustellen.

6. Dies ist ein Element span im Kontext eines Elementes p.
Hier ist die Schriftgröße 18 einzustellen.

7. Dies ist ein Element span der Klasse hintergrundnavy im Kontext eines Elementes h4.
Für alle Elemente der Klasse hintergrundnavy sind die Hintergrundfarbe auf navy, die Schriftfarbe auf weiß und die Schriftgröße auf 12pt zu setzen.

8. Dies ist ein Element span der Klasse hintergrundnavy im Kontext eines Elementes p.
Für alle Elemente der Klasse hintergrundnavy sind die Hintergrundfarbe auf navy, die Schriftfarbe auf weiß und die Schriftgröße auf 12pt zu setzen.
Beachte: Welche Deklaration hat Vorrang: Kontext oder Klasse?

9. Dies ist ein Element span der Klasse hintergrundnavy im Kontext des Elementes p.
Das Element span hat die id="span10"
Für das Element mit der id="span10" ist die Schriftgröße auf 8pt zu setzen.
Beachte: Welche Deklaration hat Vorrang: Individual oder Klasse?

10. Dies ist ein Element h3 der Klasse rot.

11. Dies ist ein Element im Kontext eines Elements h3.
Das Element h3 ist von der Klasse hintergrundnavy.
Für diese Konstellation ist als Schriftgröße ist 14pt einzustellen.

12. Dies ist ein Element span der Klasse rot im Kontext des Elements p.
Das Element span hat die id="span12".
Wenn das Element span mit der id="span12" als Klasse rot hat, dann ist die Schriftgröße auf 10pt setzen.
